# Дипломная работа к профессии Frontend-разработчик «Система бронирования ж/д билетов»

Routs:

/ – главная
/trains – результаты поиска (выбор поезда)
/seats – выбор мест (после выбора поезда)
/passengers – заполнение данных пассажиров
/payment – оплата
/confirmation – подтверждение (промежуточный экран, возможно, его нет в макетах, но можно добавить)
/success – успешный заказ
